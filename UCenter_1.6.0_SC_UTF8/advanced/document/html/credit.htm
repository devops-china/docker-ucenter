积分兑换请求

bool uc_credit_exchange_request(integer uid , integer from , integer to , integer toappid , integer amount)

函数参数
参数 含义
integer uid 用户 ID
integer from 原积分
integer to 目标积分
integer toappid 目标应用ID
integer amount 积分数额
返回值
值 含义
bool 1:请求成功
0:请求失败

接口流程

兑换参数设置
UCenter 积分兑换发起请求 API 接口 getcreditsettings 返回给 UCenter 积分项目 UCenter 中设置积分兑换参数 API 接口 updatecreditsettings 保存兑换参数
积分兑换
积分兑换界面 接口函数 uc_credit_exchange_request() 向目标应用发起兑换请求 目标应用 API 接口 updatecredit 更新用户积分 更新本应用用户积分
